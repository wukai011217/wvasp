# VASP 测试数据集

这个目录包含了用于测试和演示的真实VASP计算数据文件。

## 📁 数据文件说明

### 🔬 **主要VASP输出文件**
- `OUTCAR` - VASP主要输出文件 (606KB)
- `DOSCAR_dos` - 态密度数据文件 (2.4MB)
- `EIGENVAL` - 本征值文件 (30KB)
- `CHGCAR` - 电荷密度文件 (7.2MB)
- `CHGCAR_diff` - 电荷密度差分文件 (7.1MB)
- `LOCPOT` - 局域势文件 (13.6MB)
- `AECCAR0/AECCAR2` - 全电子电荷密度文件 (3.6MB each)
- `XDATCAR` - 分子动力学轨迹文件 (110KB)

### 📋 **结构文件**
- `CONTCAR` - 优化后的结构文件 (1.7KB)
- `CONTCAR_dos` - DOS计算的结构文件 (907B)
- `CONTCAR_fix` - 固定原子的结构文件 (18.9KB)
- `POSCAR_FS` - 最终态结构文件 (24.6KB)
- `POSCAR_IS` - 初始态结构文件 (12.3KB)
- `POSCAR_FS_sort` - 排序后的最终态结构 (11.5KB)
- `POSCAR_IS_sort` - 排序后的初始态结构 (11.5KB)

### ⚙️ **输入文件**
- `INCAR_file` - VASP输入参数文件 (840B)

### 📊 **其他数据文件**
- `methane.xsd` - 甲烷分子结构文件 (8.9KB)
- `CuO-HAc.cell` - 铜氧化物结构文件 (48.8KB)

### 📈 **绘图配置文件**
- `plot_*.json` - 各种绘图配置文件
- `plot_dos.json` - DOS绘图配置
- `plot_band.json` - 能带绘图配置
- `plot_ep.json` - 电子性质绘图配置
- `plot_pes.json` - 势能面绘图配置

## 📂 **子目录**

### `band_kpath/`
能带计算的K点路径数据

### `continuous/`
连续计算相关数据

### `dipole_input/`
偶极矩计算输入数据

### `dos/`
态密度计算专用数据（空目录）

### `electrostatic/`
静电势计算数据

### `neb/`
过渡态搜索(NEB)计算数据

### `plot_center/`
绘图中心化相关数据

### `pot/`
赝势文件目录（空目录）

### `xsd_files/`
XSD格式的结构文件

## 🎯 **使用说明**

这些数据文件可以用于：

1. **功能测试** - 验证VASP文件读取和解析功能
2. **性能测试** - 测试大文件处理性能
3. **演示示例** - 展示真实计算案例
4. **开发调试** - 调试分析算法

## 📝 **数据来源**

这些数据来自真实的VASP计算，包括：
- 结构优化计算
- 态密度计算  
- 能带结构计算
- 分子动力学模拟
- 过渡态搜索

## ⚠️ **注意事项**

- 这些是测试数据，仅用于开发和测试目的
- 文件大小从几百字节到十几MB不等
- 某些大文件可能需要较长的处理时间
- 请勿将这些数据用于实际科研发表
